
### Root 端末 (Magisk)
1. [**LSPosed for JingMatrix**](https://github.com/JingMatrix/LSPatch/releases) をインストール
2. LI**N**E アプリと LI**M**Es アプリを[こちら](https://github.com/areteruhiro/LIMEs/releases/latest)からダウンロードし、両方ともインストール
3. Google Play ストアの自動アップデートを防ぐために、[**Update Locker**](https://github.com/Xposed-Modules-Repo/ru.mike.updatelocker) や [**Hide My Applist**](https://github.com/Dr-TSNG/Hide-My-Applist) で LINE アプリを指定する<br>
  [Aurora Store](https://auroraoss.com) を使用している場合はブラックリストを使用<br>
  ※現在は、LIMEsの機能改善および追加を優先しているため、LINEの最新バージョンには対応していないこともあります。<br>
  [最新のリリース](https://github.com/areteruhiro/LIMEs/releases/latest)を確認して推奨バージョンをインストールしてください。 
5. LSPosed のモジュールから LIMEs に移動し、<kbd>モジュールの有効化</kbd> と LINE アプリにチェックを入れる
6. LINEにログインしていない場合はログインする。
7. トーク左上のSetting Directoryを開き、LIMEsの設定を保存するストレージを選択する。
8. LIMEs設定はホーム左上の⚙、もしくはLINEラボ右上よりアクセスできます。
